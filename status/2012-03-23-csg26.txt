generate image files

```Python
#!/usr/bin/python

import Image
import numpy

X = 1024
Y = 768
filename = '/tmp/x.png'

ima = numpy.zeros((Y,X,4), dtype=numpy.uint8)
ima[:,:,3] = 255 # make opaque
Image.fromarray(ima,mode='RGBA').save(filename)
```
