Comparing Compiled Languages

/* hello.c */
int main() {
    printf("hello world\n");
    return 0;
}

that was c. and here's cpp.

/* hello.cpp */
#include <iostream>
using namespace std;
int main() {
  cout << "hello world\n";
  return 0;
}

And here's java:

/* hello.java */
public class hello {
    public static void main(String[] args) {
        System.out.println("hello world");
    }
}

And compile, run, list, and time:
